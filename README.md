# IMAGE UPLOADER API
Image uploader API - challenge [https://devchallenges.io](https://devchallenges.io)

## Build with

- NodeJs (Express, Typescript)
- File storage in Local
- File manage with multer

## Getting started

### Install dependencies

```
npm install
```

### Development

```
npm run dev
```

### Production

```
npm run build
```
